ДЛЯ КОГО та КОЛИ?

  • Для професійних спортсменів, які регулярно піддають м’язи інтенсивним навантаженням

    Після тренування або фізичної активності, коли з’являється м’язова кріпатура чи дискомфорт

  • Для новачків у спорті, які тільки починають тренуватися й стикаються з болем у м’язах

    На ранок після першого чи незвичного тренування, коли з’являється м’язовий біль.

  • Для бігунів, велосипедистів та кросфітерів, після змагань або довгих сесій.

    У перші години після інтенсивного фізичного навантаження для швидшого відновлення.

  • Для людей з активним способом життя, які хочуть зберігати рухливість і комфорт

    Увечері після робочого дня, тренування або походу, щоб зняти напругу з м’язів перед сном.
